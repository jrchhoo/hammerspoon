# Hammerspoon 配置
### 目录结构
```
- Spoons
    -- CountDown.spoon   (计数，倒数器，未完成)
- modules
    -- auto_input_method.lua    （切换应用是自动切换输入法）
    -- clipboard.lua            （剪切板，记录复制文本历史）
    -- hotkey.lua               （热键，按需添加）
    -- reload.lua               （配置更改后自动加载）
    -- window.lua               （窗口管理工具，左右/上下/三栏分屏）
```
